# Trump — Submission Materials

Everything Apple needs to accept a TestFlight upload and, later, a full App Store release.

## Files

| File | Purpose |
|---|---|
| `app-store-metadata.md` | All text fields (name, description, keywords, review notes, etc.) ready to copy into App Store Connect |
| `privacy-policy.html` | Privacy policy you must host at a public URL |
| `support.html` | Support page with FAQ and contact info |

## What you still need to do manually

### 1. Host privacy policy and support page

Apple requires a public **Privacy Policy URL**. Easiest free option is GitHub Pages.

```bash
# One-time setup
gh repo create trump-card-game-web --public --add-readme
cd trump-card-game-web
cp ../trump-mobile-card-game/marketing/privacy-policy.html .
cp ../trump-mobile-card-game/marketing/support.html .
git add . && git commit -m "initial site"
git push
gh api -X POST repos/:owner/trump-card-game-web/pages -f source[branch]=main -f source[path]=/
```

Your URLs become:
- Privacy: `https://YOUR-USERNAME.github.io/trump-card-game-web/privacy-policy.html`
- Support: `https://YOUR-USERNAME.github.io/trump-card-game-web/support.html`

Paste both into App Store Connect → App Information.

### 2. Create the app record in App Store Connect

https://appstoreconnect.apple.com → **My Apps** → **+** → **New App**

Fill from `app-store-metadata.md` → App Information section.

### 3. Upload the IPA

Build the IPA with the App Store Connect build-number bump:

```bash
export ASC_KEY_ID="YOUR_KEY_ID"
export ASC_ISSUER_ID="YOUR_ISSUER_ID"
export ASC_PRIVATE_KEY_PATH="$HOME/AuthKey_YOUR_KEY_ID.p8"
scripts/ios_build/build_appstore_ipa.sh
```

Use Transporter (Mac App Store) → drag `build/ipa-appstore/Trump.ipa` → **Deliver**.

Wait ~10 min for processing.

### 4. Fill in all metadata fields

In App Store Connect, go through:
- App Information → paste name, subtitle, categories, copyright
- Pricing and Availability → Free, all countries
- App Privacy → follow the guidance in `app-store-metadata.md`
- Age Rating → answer questionnaire → 4+
- **TestFlight** tab:
  - **Test Information** → paste Beta App Description, feedback email, privacy policy URL
  - **Internal Testing** → create group, add team members (instant access)
  - **External Testing** → create group, attach build, **Submit for Beta Review**
    (paste the review notes from `app-store-metadata.md`)
  - Once approved → toggle **Public Link** → share the URL

### 5. Screenshots (required before full App Store release — NOT required for TestFlight)

You'll need screenshots at these sizes for the full App Store submission:

| Device | Size (portrait, px) | Minimum # |
|---|---|---|
| iPhone 6.9" (16 Pro Max) | 1320 × 2868 | 3 |
| iPad 13" (M4 iPad Pro) | 2064 × 2752 | 3 |

**How to capture them:**
- Deploy the TestFlight build to a real device
- Use iOS screenshot (Volume Up + Side button)
- AirDrop to your Mac
- Drag into App Store Connect → Screenshots section

(Alternatively, use Xcode's **Window → Devices and Simulators** → take screenshot from a real device paired via USB. iOS Simulator is disabled in this project because Godot doesn't ship an arm64 simulator slice.)

## Quick reference — icon

The app icon is regenerated by `../build/make_icon.swift`:

```bash
swift build/make_icon.swift build/icon-1024.png
cp build/icon-1024.png Trump/Images.xcassets/AppIcon.appiconset/Icon-1024.png
for s in 40 58 60 76 80 87 114 120 128 136 152 167 180 192; do
  sips -z $s $s build/icon-1024.png --out "Trump/Images.xcassets/AppIcon.appiconset/Icon-$s.png"
done
cp Trump/Images.xcassets/AppIcon.appiconset/Icon-120.png Trump/Images.xcassets/AppIcon.appiconset/Icon-120-1.png
```

Then re-archive and re-export:

```bash
rm -rf build/Trump.xcarchive build/ipa-appstore
xcodebuild -project Trump.xcodeproj -scheme Trump -configuration Release \
  -destination 'generic/platform=iOS' -archivePath build/Trump.xcarchive archive
xcodebuild -exportArchive -archivePath build/Trump.xcarchive \
  -exportPath build/ipa-appstore \
  -exportOptionsPlist Trump/export_options.plist \
  -allowProvisioningUpdates
```
